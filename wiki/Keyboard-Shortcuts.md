# Keyboard Shortcuts

## File

| Shortcut | Action |
|---|---|
| Ctrl+O | Import file (MIDI, MusicXML, AHK) |
| Ctrl+S | Save AHK file |

## Editing

| Shortcut | Action |
|---|---|
| Ctrl+Z | Undo |
| Ctrl+Y / Ctrl+Shift+Z | Redo |
| Ctrl+C | Copy selected notes |
| Ctrl+V | Paste notes |
| Ctrl+A | Select all visible notes |
| Escape | Deselect all / clear range |
| Del | Delete selected notes |
| Ctrl+D | Toggle draw mode (✏ Draw / ⬚ Select) |

## View

| Shortcut | Action |
|---|---|
| Ctrl+= | Zoom in |
| Ctrl+- | Zoom out |
| Home | Scroll to start of song |
| End | Scroll to end of song |

## Playback

| Shortcut | Action |
|---|---|
| Space | Play / Pause |
| Click minimap | Scroll to position |

## Mouse Controls

| Action | Effect |
|---|---|
| Click note | Select note |
| Ctrl+Click note | Add/remove note from selection |
| Drag note | Move selected notes |
| Drag right edge of note | Resize (change duration) |
| Click empty area + drag | Box select |
| Click ruler + drag | Select time range |
| Right-click ruler | Trim menu |
| Right-click note | Delete note |
| Mouse wheel | Scroll vertically |
| Shift + wheel | Scroll horizontally |
| Ctrl + wheel | Zoom in/out |
| Ctrl+Shift+Click note | Toggle manual simplified override (on ✂ tracks) |

## Draw Mode

| Action | Effect |
|---|---|
| Click (draw mode) | Place a new note |
| Click + drag (draw mode) | Place a note with custom duration |
